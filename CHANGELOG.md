## [v0.1.0] - initial project setup

### Added
- Initial project setup for reasoning-pricer
- Core Python module structure with src/main.py and src/models.py
- Data management system with JSON file storage (data/1.json through data/10.json)
- Git version control configuration (.gitignore)
- Project documentation (README.md)

### Project Structure
- **src/**: Main Python source code directory
  - `main.py`: Entry point and main application logic
  - `models.py`: Data models and business logic
- **data/**: Data storage directory
  - Contains 10 JSON files (1.json - 10.json) for storing pricing/reasoning data
- **README.md**: Project documentation and setup instructions
- **.gitignore**: Git ignore configuration for Python projects

### Technical Details
- Python-based project for reasoning and pricing functionality
- JSON-based data storage system
- Modular code structure separating main logic from data models
- Version control ready with appropriate .gitignore configuration
